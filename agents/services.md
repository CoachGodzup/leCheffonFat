# Services

## API / Data

| File                                   | Description                                                                                                                                                                                                                                                                                                                                   |
| -------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/service/meal-db-service.ts`       | TheMealDB API (via local proxy) — search meals by name, get by ID, random, categories, filter by category/area/ingredient, list areas. Also `getMealPageUrl(meal)` returns the best external URL for a meal (strSource → strYoutube → themealdb.com/meal/{id}). Set `NEXT_PUBLIC_MOCK_API=1` to use offline mock data from `meal-db-mock.ts`. |
| `src/service/meal-db-mock.ts`          | Offline mock data for TheMealDB API — used when `NEXT_PUBLIC_MOCK_API=1`. Provides fake meals, categories, and areas for development/testing.                                                                                                                                                                                                 |
| `src/app/api/meals/[...path]/route.ts` | Next.js Route Handler — proxies TheMealDB requests server-side, injects API key from `.env`. Imports `API_BASE` from `meal-db-service`.                                                                                                                                                                                                      |
| `src/constants/filter-options.tsx`     | Shared `FILTER_OPTIONS` array with ThumbsUp/ThumbsDown icons for like/dislike/unrated filtering. Used by `Sidebar` and `history/page`.                                                                                                                                                                                                        |

## Hooks

| File                                 | Description                                                                                                                                                                                                      |
| ------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/hooks/use-api.ts`               | Generic primitive — async fetcher with loading/error/data/refetch state                                                                                                                                          |
| `src/hooks/use-categories.ts`        | Wraps `getCategories()` — returns `UseApiReturn<Category[]>`                                                                                                                                                     |
| `src/hooks/use-areas-by-category.ts` | Accepts a category string, wraps `filterByCategory()`, deduplicates areas — returns `UseApiReturn<Area[]>`                                                                                                       |
| `src/hooks/use-random-meal.ts`       | Wraps `getRandomMealByFilter(category, area)` — returns `UseApiReturn<Meal>`, refetches when category/area change                                                                                                |
| `src/hooks/use-meal-by-id.ts`        | Wraps `getMealById(id)` — returns `UseApiReturn<Meal>`, refetches when id changes                                                                                                                                |
| `src/hooks/use-debounce.ts`          | Generic debounce hook — takes `value: T` and `delay: number`, returns debounced value after delay. Used by `useSearch`.                                                                                          |
| `src/hooks/use-search.ts`            | Search hook — manages input state, debounces with `useDebounce(300ms)`, calls `searchMealsByName` via `useApi` when term >= 3 chars. Returns `{ searchText, setSearchText, meals, isLoading, error, isActive }`. |
