# Services

| File                                                       | Description                                                                                                                                                                                                                                                    |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/service/meal-db-service.ts`                           | TheMealDB API (via local proxy) тАФ search meals by name, get by ID, random, categories, filter by category/area/ingredient, list areas. Also `getMealPageUrl(meal)` returns the best external URL for a meal (strSource тЖТ strYoutube тЖТ themealdb.com/meal/{id}) |
| `src/app/api/meals/[...path]/route.ts`                     | Next.js Route Handler тАФ proxies TheMealDB requests server-side, injects API key from `.secret`                                                                                                                                                                 |
| `src/hooks/use-api.ts`                                     | Generic primitive тАФ async fetcher with loading/error/data/refetch state                                                                                                                                                                                        |
| `src/hooks/use-categories.ts`                              | Wraps `getCategories()` тАФ returns `UseApiReturn<Category[]>`                                                                                                                                                                                                   |
| `src/hooks/use-areas-by-category.ts`                       | Accepts a category string, wraps `filterByCategory()`, deduplicates areas тАФ returns `UseApiReturn<Area[]>`                                                                                                                                                     |
| `src/hooks/use-random-meal.ts`                             | Wraps `getRandomMealByFilter(category, area)` тАФ returns `UseApiReturn<Meal>`, refetches when category/area change                                                                                                                                              |
| `src/hooks/use-meal-by-id.ts`                              | Wraps `getMealById(id)` тАФ returns `UseApiReturn<Meal>`, refetches when id changes                                                                                                                                                                              |
| `src/components/RecommendationView/RecommendationView.tsx` | Shared component that renders meal data, loading/error state, RecipeCtas, and LikeDislikeCtas. Used by both random and ID-based recommendation pages.                                                                                                          |
| `src/store/slices/history-slice.ts`                        | Storico raccomandazioni (calls) persistito su localStorage                                                                                                                                                                                                     |
| `src/app/components/atoms/CheckboxFilter.tsx`              | Generic checkbox filter atom тАФ takes `legend`, `options`, `value`, `onChange`. Toggles items in an array. Used by Sidebar for rating filter.                                                                                                                   |
| `src/app/components/atoms/SortBy.tsx`                     | Sort toggle button тАФ takes `value` ("asc" | "desc") and `onChange`, shows ЁЯФ╝/ЁЯФ╜.                                                                                                                                                                               |
| `src/app/components/List.tsx`                             | Card grid list for history entries тАФ takes `entries: Call[]`, renders image, title, category/area, date, and like indicator. Reusable across pages.                                                                                                            |
