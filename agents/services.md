# Services

| File                                                       | Description                                                                                                                                                                                                                                                    |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/service/meal-db-service.ts`                           | TheMealDB API (via local proxy) — search meals by name, get by ID, random, categories, filter by category/area/ingredient, list areas. Also `getMealPageUrl(meal)` returns the best external URL for a meal (strSource → strYoutube → themealdb.com/meal/{id}) |
| `src/app/api/meals/[...path]/route.ts`                     | Next.js Route Handler — proxies TheMealDB requests server-side, injects API key from `.secret`                                                                                                                                                                 |
| `src/hooks/use-api.ts`                                     | Generic primitive — async fetcher with loading/error/data/refetch state                                                                                                                                                                                        |
| `src/hooks/use-categories.ts`                              | Wraps `getCategories()` — returns `UseApiReturn<Category[]>`                                                                                                                                                                                                   |
| `src/hooks/use-areas-by-category.ts`                       | Accepts a category string, wraps `filterByCategory()`, deduplicates areas — returns `UseApiReturn<Area[]>`                                                                                                                                                     |
| `src/hooks/use-random-meal.ts`                             | Wraps `getRandomMealByFilter(category, area)` — returns `UseApiReturn<Meal>`, refetches when category/area change                                                                                                                                              |
| `src/hooks/use-meal-by-id.ts`                              | Wraps `getMealById(id)` — returns `UseApiReturn<Meal>`, refetches when id changes                                                                                                                                                                              |
| `src/hooks/use-debounce.ts`                                | Generic debounce hook — takes `value: T` and `delay: number`, returns debounced value after delay. Used by `useSearch`.                                                                                                                                        |
| `src/hooks/use-search.ts`                                  | Search hook — manages input state, debounces with `useDebounce(300ms)`, calls `searchMealsByName` via `useApi` when term >= 3 chars. Returns `{ searchText, setSearchText, meals, isLoading, error, isActive }`.                                               |
| `src/components/RecommendationView/RecommendationView.tsx` | Shared component that renders meal data, loading/error state, RecipeCtas, and LikeDislikeCtas. Used by both random and ID-based recommendation pages.                                                                                                          |
| `src/store/slices/history-slice.ts`                        | Storico raccomandazioni (calls) persistito su localStorage                                                                                                                                                                                                     |
| `src/app/components/atoms/CheckboxFilter.tsx`              | Generic checkbox filter atom — takes `legend`, `options`, `value`, `onChange`. Toggles items in an array. Used by Sidebar for rating filter.                                                                                                                   |
| `src/app/components/atoms/SortBy.tsx`                      | Sort toggle button — takes `value` ("asc" \| "desc") and `onChange`, shows 🔼/🔽.                                                                                                                                                                              |
| `src/app/components/HistoryList.tsx`                       | Card grid for history entries — takes `entries: Call[]`, renders image, title, category/area, date, and like indicator.                                                                                                                                        |
| `src/app/components/SearchResults.tsx`                     | Card grid for search results — takes `meals: Meal[]`, renders image, title, category/area, links to `/recommendation/[id]`.                                                                                                                                    |
| `src/app/search/page.tsx`                                  | Search page — controlled input, uses `useSearch`, renders loading/error/empty/result states.                                                                                                                                                                   |
